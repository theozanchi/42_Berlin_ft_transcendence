Game Instructions

To play the 3D Ponger Puff Pong, use the following keys

  • IJKL
    Move the rackets position on active plane up, left, down or right.
  • WASD
    Rotate the cube to switch the active plane.
  • SPACE
    Serve the ball if static and start the game
